### 1. Clone repository

```
git clone <repo_address>
```

### 2. Make changes
  - Edit file
  - Add new files
  
### 3. Add changed files/folders to add list

```
git add <filename>
```

For folders:

```
git add <folder>
git add <folder>/*
```

 ### 4. Commit files

```
git commit -m "commit message"
```

### 5. Push changes

```
git push -u origin master
```
